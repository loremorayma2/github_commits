## Flutter Project

### Dependencies

```
dependencies:
    ...
    flutter_svg: 2.0.9
    http: 1.2.0
    url_launcher: 6.2.3
```

### Steps

#### Clone the repository

```
git clone https://github.com/loremorayma2/github_commits.git

```

#### Install dependencies

```
cd github_commits
flutter pub get

```

#### Run the application

```
flutter run

```

#### Run tests

```
flutter test
```
